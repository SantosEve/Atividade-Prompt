Como usá-lo?


Agora que você já sabe o que é o Prompt de Comando, invista nessa aventura e descubra como desfrutar dele da melhor forma!


⚠️ Antes de continuar, saiba que:


Para ver a lista completa de comandos disponíveis no CMD, basta digitar "help" na linha de comando e pressionar Enter. Isso exibirá uma lista de todos os comandos disponíveis no CMD, juntamente com uma breve descrição de cada comando e como usá-lo. Pode-se, também, digitar "help" seguido do nome de um comando específico para obter informações mais detalhadas sobre como usar esse comando.

E para executar um comando, basta digitar o nome do comando na linha de comando e pressionar Enter.




A imagem acima é uma reprodução da internet revelando como acessar um diretório no prompt do Windows 10, mas logo abaixo você pode conhecer mais 5 comandos:

  • Ipconfig

    • Exibe informações sobre a configuração de rede do sistema, incluindo endereços IP, máscaras de sub-rede e gateways padrão.
    Além disso, é o comando responsável por exibir informações sobre todas as interfaces de rede: ipconfig /all;
    E para exibir informações resumidas sobre a interface de rede sem fio: ipconfig /wireless


  • Shutdown

    • Usado para desligar ou reiniciar o sistema.
    Desligar o sistema imediatamente: shutdown /s /t 0
    Reiniciar o sistema: shutdown /r


  • Ping

    • Usado para testar a conectividade da rede, enviando pacotes para um endereço IP e medindo o tempo de resposta.
    Para testar a conectividade com um host remoto: ping www.exemplo.com
    E para testar a conectividade com um host remoto e exibir estatísticas detalhadas: ping -n 10 www.exemplo.com


  • Netstat

    • Usado para exibir informações sobre as conexões de rede ativas e estatísticas.
    Exibir todas as conexões TCP ativas: netstat -a
    E para exibir todas as conexões UDP ativas: netstat -u


  • PathpIng

    • Usado para diagnosticar problemas de conexão de rede, exibindo informações sobre o tempo de viagem e perda de pacotes em cada roteador.


    Diagnosticar problemas de conexão de rede com um host remoto: pathping www.exemplo.com
    E para diagnosticar problemas de conexão de rede com um host remoto e exibir informações em formato gráfico: pathping -n -h 10 -q 1 -p www.exemplo.com
