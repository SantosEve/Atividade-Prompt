CMD - Command Prompt

O CMD ou Prompt de Comando é uma interface de linha de comando que capacita os usuários a interagirem com o sistema operacional. Utilizando o CMD, é possível realizar operações como criar, copiar, renomear e excluir arquivos e diretórios. Além disso, é possível gerenciar processos em execução e ajustar as configurações do sistema.


Como abrir o Prompt de Comando nos principais sistemas operacionais:
  • Windows

    • Pesquise o prompt de comando: Clique na barra de pesquisa no canto inferior esquerdo e digite “prompt de comando” ou “cmd”;
    Inicie o programa: Clique no ícone do prompt ou na opção “Abrir” no menu lateral direito.
    → Outra forma de acessar o cmd no Windows é pressionando as teclas de atalho windows (⊞) + r

  • Linux

    • Pesquise o prompt de comando: Clique na barra de pesquisa no canto inferior esquerdo e digite “prompt de comando” ou “cmd”
    Inicie o programa: Clique no ícone do prompt.

    → Além disso, há outras formas, como os atalhos do teclado, por meio do pressionamento das teclas Ctrl + Alt + T
